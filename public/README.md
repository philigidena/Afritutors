# Put your logo and assets here

## Recommended structure:
```
public/
├── logo.svg          # Main logo
├── logo-light.svg    # Light version for dark backgrounds
├── favicon.ico       # Browser favicon
├── og-image.jpg      # Social media preview (1200x630px)
├── images/
│   ├── hero/
│   ├── courses/
│   ├── instructors/
│   └── production/
└── videos/
    └── showreel.mp4
```

## Tips:
- Use SVG for logos (scalable, crisp)
- Optimize images with WebP format
- Keep videos under 10MB for fast loading
- Use 16:9 aspect ratio for hero videos
